# react-native-ios-toggle

A 1:1 modern iOS toggle component made it with Reanimated (better perf), haptics feedback and fully customizable

## Installation

```sh
npm install react-native-ios-toggle
```

## Usage

```js
import { IosToggleView } from "react-native-ios-toggle";

// ...

<IosToggleView color="tomato" />
```
